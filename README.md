# Turbo Traffic 3D 🏎

Трёхмерный бесконечный раннер для **Яндекс Игр**: уворачивайся от трафика на четырёхполосном шоссе, собирай монеты, лови пауэрапы и открывай новые машины в гараже.

![genre](https://img.shields.io/badge/genre-hypercasual%20racer-orange) ![stack](https://img.shields.io/badge/stack-Three.js%20%2B%20Vite-blue)

## Игра

- **Жанр:** гиперказуальный 3D-раннер (трафик-рейсер) — играется одной рукой, сессия 1–3 минуты.
- **Управление:** на телефоне — свайпы влево/вправо; на десктопе — `← →` или `A / D`, пауза — `Esc` / `P`.
- **Механики:**
  - скорость растёт со временем, трафик плотнеет;
  - генератор трафика гарантирует проходимый маршрут («безопасная полоса» блуждает по дороге);
  - монеты, бонус «на волоске» за проезд впритирку, сбиваемые конусы;
  - пауэрапы: 🧲 магнит, 🛡 щит, ✖2 очки;
  - **гараж** — 12 машин, покупаются за монеты (мета-прогрессия для удержания);
  - **продолжение за рекламу** после аварии (rewarded video, 1 раз за забег).
- **Локализация:** русский, английский, турецкий — язык берётся из SDK.
- **Звук:** полностью процедурный WebAudio (двигатель, монеты, авария, фоновый синт-пад) — в билде нет ни одного аудиофайла.

## Интеграция с Яндекс Играми

Все обязательные требования площадки выполнены:

| Требование | Где реализовано |
|---|---|
| Подключение SDK по относительному пути `/sdk.js` | `src/yandex.js` (при сбое игра работает автономно) |
| `LoadingAPI.ready()` после полной загрузки | `src/main.js → boot()` |
| `GameplayAPI.start()/stop()` на старте/паузе/аварии/меню | `src/main.js`, хуки в `src/game.js` |
| Interstitial-реклама только в естественных паузах, с кулдауном 75 с | `src/yandex.js → showInterstitial()` |
| Rewarded video за «продолжить» | `src/yandex.js → showRewarded()` |
| Пауза игры и звука при показе рекламы и сворачивании вкладки | `adCallbacks` + `visibilitychange` в `src/main.js` |
| Облачные сохранения (`player.setData`) + fallback на `localStorage` | `src/save.js` |
| Лидерборд `score` | `src/yandex.js → submitScore()` |
| Язык из `ysdk.environment.i18n.lang` | `src/i18n.js` |
| Мобильные и десктоп, обе ориентации | адаптивный FOV/камера в `src/world.js`, `src/game.js` |
| Нет внешних запросов, относительные пути (`base: './'`) | `vite.config.js`, все ассеты в билде |
| Имена файлов без пробелов и кириллицы, `index.html` в корне | структура `dist/` |
| Автоснижение качества при низком FPS (слабые устройства) | `src/main.js → tick()` |

## Разработка

```bash
npm install
npm run dev      # локальная разработка (SDK замокан — реклама «показывается» мгновенно)
npm run build    # production-билд в dist/
npm run zip      # dist → turbo-traffic-3d.zip для загрузки в консоль Яндекс Игр
```

## Публикация в Яндекс Играх

1. `npm run zip` → получится `turbo-traffic-3d.zip` (≈3.5 МБ, лимит — 100 МБ).
2. В [консоли разработчика](https://games.yandex.ru/console) создайте черновик игры и загрузите архив.
3. Создайте **лидерборд** с техническим именем `score` (тип — «максимальное значение»).
4. Укажите поддержку: десктоп + мобильные, обе ориентации.
5. Проверьте интеграцию через `?draft=true&debug-mode=16` — в лоадере должен появиться индикатор SDK, а в консоли отладки — события `LoadingAPI.ready` и `GameplayAPI`.

## Структура

```
index.html            разметка всех экранов (меню, HUD, гараж, пауза, геймовер)
public/assets/models/ GLB-модели (CC0)
src/
  main.js             точка входа: загрузка, игровой цикл, ввод, жизненный цикл вкладки
  game.js             геймплей: игрок, трафик, коллизии, монеты, пауэрапы, камера
  world.js            сцена: дорога, разметка, обочины, освещение, небо
  assets.js           загрузка и нормализация GLB
  ui.js               экраны, HUD, гараж с 3D-превью машин
  yandex.js           обёртка Yandex Games SDK (реклама, сейвы, лидерборд)
  audio.js            процедурный звук (WebAudio)
  i18n.js             локализация ru/en/tr
  save.js             прогресс: облако + localStorage
```

## Ассеты и лицензии

3D-модели — [Kenney](https://kenney.nl) **Car Kit** и **Nature Kit**, лицензия **CC0 1.0** (public domain, указание автора не требуется, но Kenney — красавчик). Шрифты системные, звук процедурный, текстур и аудиофайлов в проекте нет.
