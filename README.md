# Turbo Traffic 3D 🏎 («Турбо Трафик 3D»)

Трёхмерный трафик-рейсер для **Яндекс Игр**: три режима, встречная полоса, нитро с тараном, цикл день/ночь и дождь, гараж на 12 машин, 18 достижений, 12 рангов игрока, задания, ежедневные бонусы и куча пасхалок.

![genre](https://img.shields.io/badge/genre-hypercasual%20racer-orange) ![stack](https://img.shields.io/badge/stack-Three.js%20%2B%20Vite-blue)

## Режимы

| Режим | Суть |
|---|---|
| **Классика** | Бесконечный заезд, скорость растёт, с 20-й секунды открывается встречная полоса. Продолжение за rewarded-рекламу |
| **Спринт** | 90 секунд на максимум очков, высокая скорость и встречка сразу |
| **Разгром** | 60 секунд неуязвимости: тарань всех, очки и монеты за каждый снесённый автомобиль |

## Механики

- **Встречная полоса** — левая полоса с лобовым трафиком: «на волоске» там даёт x2 очков, туда же подкидывает монеты;
- **Нитро** — шкала копится за проезды впритирку и монеты; при активации x1.5 скорости и таран вместо аварии;
- **Комбо** — цепочки «на волоске» умножают бонус (x2, x3, …), счётчик прямо на HUD;
- **Достижения** — 18 наград с прогресс-барами и монетами (экран «Награды»);
- **Ранги** — 12 званий от Новичка до Бога дороги: XP за очки, прогресс-бар в меню, монеты за новый ранг;
- **Цикл день/ночь** — закат → ночь со звёздами и фарами → рассвет → день, каждые ~2 минуты; ~каждый пятый заезд идёт под дождём;
- **Пауэрапы**: 🧲 магнит, 🛡 щит, ✖2 очки; сбиваемые конусы; гудок, от которого трафик шарахается;
- **Гараж** — 12 машин Kenney за монеты; **задания** (3 активных, растущие цели); **ежедневный бонус** со стриком;
- **Сирены с эффектом Доплера** у полиции/скорой/пожарной в трафике, мосты над трассой, дым из-под колёс, конфетти на рекорде, вибрация на телефонах;
- **Пасхалки**: НЛО ночью (+100 монет), конами-код (или 7 тапов по логотипу), фразы-подколы после аварии, шутки при покупке некоторых машин.

## Интеграция с Яндекс Играми

| Требование | Где реализовано |
|---|---|
| SDK по относительному пути `/sdk.js`, graceful fallback | `src/yandex.js` |
| `LoadingAPI.ready()` после загрузки | `src/main.js → boot()` |
| `GameplayAPI.start()/stop()` на всех переходах | `src/main.js`, хуки `src/game.js` |
| Interstitial в естественных паузах, кулдаун 75 с | `src/yandex.js → showInterstitial()` |
| Rewarded: продолжение, x2 монет, +250 в гараже | `src/main.js → bindUI()` |
| **Sticky-баннер**: показ в меню, скрытие в геймплее | `showBanner()/hideBanner()` |
| Пауза игры и звука при рекламе/сворачивании | `adCallbacks` + `visibilitychange` + `blur` |
| Облачные сохранения + localStorage fallback | `src/save.js` |
| Лидерборд `score`: запись и топ-10 в меню | `src/yandex.js` |
| Язык из `ysdk.environment.i18n.lang` (ru/en) | `src/i18n.js` |
| Мобильные + десктоп, обе ориентации, свайпы | `src/main.js`, адаптивная камера |
| Нет внешних запросов, относительные пути | `vite.config.js` (`base: './'`) |
| Автоснижение качества + ручной переключатель | `src/main.js`, настройки |

## Управление

- **Десктоп**: ← → / A D — руль, W / ↑ / Space — нитро, G — гудок, Esc — пауза;
- **Мобильные**: тап по краям экрана или свайпы — руль, свайп вверх или кнопка 🔥 — нитро, кнопка 📢 — гудок.

## Разработка

```bash
npm install
npm run dev      # локально: SDK замокан, rewarded «выдаёт награду» сразу
npm run build    # production-билд в dist/
npm run zip      # turbo-traffic-3d.zip для консоли Яндекс Игр
```

## Публикация

1. `npm run zip` → архив ≈6 МБ (лимит 100 МБ).
2. В [консоли разработчика](https://games.yandex.ru/console) создайте черновик и загрузите архив.
3. Создайте **лидерборд** `score` (тип «максимум»).
4. Иконки и скриншоты для карточки игры — в `promo/` (иконка 512/256/128, 6 скриншотов).
5. Проверка: `?draft=true&debug-mode=16` — индикатор SDK в лоадере, события Loading/Gameplay API в консоли отладки.

## Структура

```
index.html            все экраны: меню, HUD, гараж, задания, рекорды, настройки, пауза, геймовер
public/assets/models/ GLB-модели (CC0)
public/assets/audio/  музыка и звуки (CC0)
promo/                иконки и скриншоты для консоли Яндекса (в архив не входят)
src/
  main.js             вход: SDK, загрузка, цикл, ввод, реклама, daily-бонус, конами-код
  game.js             геймплей: режимы, трафик, встречка, нитро, комбо, сирены, НЛО
  world.js            сцена: дорога, день/ночь, звёзды, мосты, освещение
  missions.js         задания с прогрессией
  ui.js               экраны, HUD, гараж, конфетти
  yandex.js           SDK: реклама (interstitial/rewarded/sticky), сейвы, лидерборд
  audio.js            музыка из файлов + процедурные SFX, сирены с Доплером
  i18n.js             локализация ru/en
  save.js             прогресс: облако + localStorage
```
